## 🌟 特性

- **苹果风格设计**: 极简主义、精致细节、优雅排版
- **响应式布局**: 完美适配桌面、平板、手机等所有设备
- **深色模式**: 自动适配系统主题，提供舒适的夜间浏览体验
- **高性能**: 优化的CSS和JavaScript，快速加载
- **可访问性**: 符合WCAG标准，支持键盘导航和屏幕阅读器
- **语义化HTML**: 清晰的结构，良好的SEO基础

## 🚀 快速开始

### 克隆仓库
```bash
git clone https://gitcode.com/zackal/AwesomeIntelligence.git
cd AwesomeIntelligence
```

### 本地预览
直接在浏览器中打开 `index.html` 文件即可预览网站。

### 自定义
- **修改内容**: 编辑 `index.html` 中的文本内容
- **调整样式**: 修改 `css/style.css` 和 `css/responsive.css`
- **添加功能**: 在 `js/` 目录中添加新的JavaScript文件

## 📁 项目结构

```
AwesomeIntelligence/
```

## 🎨 设计规范

### 色彩系统
- **主色**: `#0071e3` (Apple Blue)
- **背景**: `#ffffff` / `#f5f5f7` (Light Mode), `#000000` / `#121212` (Dark Mode)
- **文字**: `#000000` / `#86868b` (Light Mode), `#ffffff` / `#aeaeb2` (Dark Mode)

### 字体
- **字体栈**: `-apple-system, BlinkMacSystemFont, 'SF Pro Text', 'Helvetica Neue', sans-serif`
- **标题字体**: `'SF Pro Display'` for better hierarchy

### 间距系统
使用标准的8px网格系统：4px, 8px, 16px, 24px, 32px, 48px, 64px, 96px

## 📱 响应式断点

- **桌面端**: ≥ 769px
- **平板端**: ≤ 768px
- **手机端**: ≤ 480px
- **超小屏**: ≤ 320px
